It's the tinininiest!

Removes all spaces from a string

## Install

>$ npm install @raylai/tiny

## Usage
const tiny = require("@raylai/tiny");

tiny("so much space!a hhhh")

//=>"somuchspace!ahhhh"


tiny(1337)

//=> error
